Command Compositions

Glue verbs into
sentences.

One command is an action. Combine them - in order, all at once, or racing to the finish - and you describe a whole routine. Watch each rule on a timeline.

Real behaviors are several commands woven together.

"Drive to the goal, then shoot." "Aim while spinning up." WPILib gives you composition operators to express exactly that. Pick one below and play it - three commands (A=2 s, B=3 s, C=1 s) run, and the whole group ends by that rule.

Green caps = the command finished on its own. Amber caps = it was cut short because the group's rule ended first.

In code

The four operators

Each composition has a factory (and a fluent method). They take any commands and return a new command you can schedule like any other:

Composing commands Java · WPILib
Commands.sequence(a, b, c);    // one after another; ends when the last finishes
Commands.parallel(a, b, c);    // all at once; ends when ALL finish
Commands.deadline(a, b, c);    // all at once; ends when A (the deadline) finishes
Commands.race(a, b, c);        // all at once; ends when ANY finishes

a.andThen(b);          // fluent sequence: a, then b
aim.alongWith(spinUp);   // fluent parallel
drive.withTimeout(3);   // race against a 3-second timer
withTimeout is just a race

command.withTimeout(3) races your command against a 3-second timer - whichever finishes first ends the group. It's the everyday way to say "do this, but give up after 3 seconds," and it keeps an autonomous routine from hanging forever.

Checkpoint

Quick check

Pick an answer to see why.

Question 1
A parallel composition of A, B, C ends when…
Parallel runs them together and waits for every one to complete. (Race ends when the first finishes; deadline ends when the chosen deadline command finishes.)
Question 2
You want "aim, but give up after 2 seconds." Which fits?
withTimeout(2) races aim against a 2-second timer - whichever ends first stops the group. It's the standard "do it, but don't hang" pattern.
Question 3
In a race(A, B, C), when does the whole group end?
A race ends the moment any member finishes; the others are interrupted (and their end(true) runs). Useful for "shoot OR time out," for example.
