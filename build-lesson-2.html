Hand Tools

The right tool
for the job.

Most of building a robot is a handful of simple hand tools used well. Knowing which one to reach for, and using it the right way, is half the battle. Forcing the wrong tool rounds off bolts and bloodies knuckles.

A small kit covers almost everything.

Drive a bolt, cut a tube, smooth an edge, take a measurement. Each job has a tool that makes it easy and a dozen that make it hard. Try the matcher: read the job, pick the tool you'd grab.

Match the size, every time

A driver one size too small spins inside the head and rounds it out, a stripped bolt is a real headache to remove. Seat the tool fully and use the size that fits snugly. The same goes for metric versus imperial: an SAE socket on a metric bolt almost fits, and "almost" is how you strip it. FRC mixes both, so keep your sets separate and labeled.

Going further · optional

An already-stripped hex socket is not always a dead end. A torx bit one size up, or a hex key with a drop of valve-grinding paste for extra bite, can sometimes break a rounded screw loose. Try this thought experiment: a ball-end hex key is faster because you can drive a screw at an angle, so why would a competitive team still reach for a plain straight key when torquing a bolt down hard?

Checkpoint

Quick check

Pick an answer to see why.

Question 1
What drives a socket-head cap screw, the most common FRC bolt?
Socket-head cap screws have a hexagonal socket in the top, so a hex (Allen) key fits inside and drives them. It's the tool you'll use more than any other.
Question 2
Why match the driver size carefully to the fastener?
A tool that's too small can't grip, so it rounds out the head. A stripped fastener is slow and frustrating to remove. Seat the right size fully before you turn.
Question 3
You just cut a tube and the edge is sharp. What do you reach for?
Every cut and drilled hole leaves a sharp burr. A quick pass with a file or deburring tool removes it so it won't slice a hand or interfere with assembly.
